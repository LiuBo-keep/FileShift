# FileShift

🚀 **FileShift** 是一款基于 **Java 21 + JavaFX** 的跨平台桌面文件转换工具，专注于 **PDF / Word / Excel / 图片** 等常见文档格式的高效互转，目标是提供 **像真正商业软件一样的体验**。

------

## 📦 应用性质

**FileShift 是一款永久免费（Free）的桌面文件转换应用。**

- ✅ 永久免费使用
- 🚫 无广告、无内购
- 🔒 不上传文件，全部本地处理
- 👤 不强制登录、不收集隐私数据

------

## ✨ 产品定位

- 🎯 **本地桌面应用**（非网页、非云端）
- 🖥 **跨平台**：Windows / macOS / Linux
- 🔒 **隐私友好**：文件不上传服务器，全部本地处理
- ⚡ **高效率**：支持批量文件、拖拽操作、状态可视化

------

## 🧩 核心功能

### 已规划 / 已实现

- 📂 文件拖拽到列表（TableView）
- 📊 文件状态展示：
  - ⏳ 等待中（WAITING）
  - ⚙ 转换中（RUNNING）
  - ✅ 已完成（DONE）
- 🎛 左侧功能导航（带选中态）
- ▶ 一键开始转换
- 📈 全局进度条显示
- 📁 输出目录配置
- 🌙 深色 / 浅色主题切换
- 📦 jpackage 打包为系统安装包
- 📦 删除指定无用文件功能

### 转换能力（规划）

- PDF → Word
- PDF → Excel
- Word → PDF
- Excel → PDF
- Images → PDF

------

## 🖼 UI 设计理念

- 左侧功能栏 + 右侧主内容区（专业工具布局）
- 强调「拖进去就能用」的低学习成本
- 状态、进度、反馈 **始终可见**
- 风格克制、偏工具类而非花哨设计

## ▶ 运行方式

### 环境要求

- JDK 21+
- Maven 3.9+

### 启动项目

```bash
mvn clean javafx:run
```
